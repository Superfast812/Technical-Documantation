Spacing_With_Flexboxes

One of the best ways to order items is using "display: flex" in the CSS code.

To add space between the items, use the "gap:" element with a pixel value.

It might be best to set a width: element to your items.

The overall flexbox can be positioned using the margin: element.

Background_Colors

To set background colors, use the background-color: element.

To set the color, type a 6 digit # value like #ffffff and the color will appear.

To add addition background color space, use the padding: element.

Alternatively, you can use the border: property assigned the same color.

Z-Index

There is padding-left which adds space of the left.

There is padding-right which adds space of the right.

There is padding-top which adds space of the top.

There is padding-bottom which adds space of the bottom.
